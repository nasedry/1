# База даних для туристичних маршрутів та екскурсій (Курсовий проєкт)

## Огляд проєкту
Це документація та код до моєї курсової роботи з "Баз даних". Проєкт являє собою реляційну базу даних, створену для автоматизації роботи туристичного агентства. Дозволяє керувати турами, маршрутами, клієнтами та бронюваннями.

## Технології
* **База даних:** MySQL 8.0+
* **Мова програмування:** Python 3.10+
* **Веб-фреймворк:** Flask (для демонстрації)

## Структура документації
Тут є всі файли, які вимагалися для практичної роботи:

| Файл | Опис |
| :--- | :--- |
| `SCHEMA.sql` | Схема бази даних (всі команди CREATE TABLE, індекси, тригери). |
| `VISION.md` | Проблема, цілі та аудиторія проєкту. |
| `REQUIREMENTS.md` | Функціональні та Нефункціональні вимоги. |
| `DEPLOYMENT.md` | Інструкція, як запустити базу та веб-додаток. |
| `README.md` | Загальний огляд (цей файл). |

## Як запустити
Потрібно виконати два кроки:

1.  **Запустити базу даних:** Інструкції є у файлі **`DEPLOYMENT.md`**.
2.  **Запустити Flask-додаток:** Після налаштування бази даних, запустити додаток командою `flask run`.
