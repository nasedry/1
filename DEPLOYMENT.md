# Інструкції з розгортання

## 1. Що потрібно мати (Вимоги)
* Встановлений **MySQL Server** (версії 8.0 або новіше).
* Встановлений **Python** (версії 3.10 або новіше).
* Програму **Git** для клонування репозиторію.

## 2. Як запустити базу даних
Мені потрібно виконати два SQL-скрипти, щоб створити базу і заповнити її даними:

1.  **Створюємо структуру (Таблиці):**
    ```bash
    mysql -u root -p < SCHEMA.sql
    ```
    (Тут `SCHEMA.sql` — це мій файл зі створення таблиць)
2.  **Заповнюємо даними (Тестові тури, клієнти):**
    ```bash
    mysql -u root -p < tourism_db_3_1_3_2.sql
    ```
    (Тут `tourism_db_3_1_3_2.sql` — це мій файл з INSERT-запитами)

## 3. Як запустити веб-додаток (Flask)
Для демонстрації я зробила простий додаток на Flask.

1.  **Налаштування:** Потрібно скопіювати файл `.env.example` у файл `.env` та вказати свій пароль до бази MySQL.
    ```bash
    cp .env.example .env
    ```
2.  **Встановлення залежностей:** Створити віртуальне оточення і встановити всі необхідні бібліотеки з `requirements.txt`.
    ```bash
    python -m venv .venv
    source .venv/bin/activate 
    pip install -r requirements.txt
    ```
3.  **Запуск:**
    ```bash
    export FLASK_APP=app.py 
    flask run
    # Додаток запуститься за адресою [http://127.0.0.1:5000](http://127.0.0.1:5000)
    ```
